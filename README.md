# -chrome-
本插件（豆瓣电影阅读标记数据导出助手）所有数据处理均在用户本地浏览器完成，不会将用户的 Cookies、图书数据或任何个人信息上传至任何第三方服务器。插件仅在用户主动点击时运行。

# 使用方法
1.下载项目
2.打开chrome浏览器。设置-插件-已解压的扩展程序，选择解压的文件夹。
